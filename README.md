РюЁ рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃўрЃА рЃЊрЃљрЃАрЃбрЃљрЃарЃбрЃЋрЃљ:
    Рђб РюЁ npm install
    Рђб РюЁ npm run start 
    Рђб РюЁ npm run mock-server

РюЁ рЃерЃћрЃАрЃарЃБрЃџрЃћрЃЉрЃБрЃџрЃў рЃцрЃБрЃюрЃЦрЃфрЃўрЃЮрЃюрЃљрЃџрЃў
1. рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃўрЃџрЃўрЃА рЃЊрЃћрЃерЃЉрЃЮрЃарЃЊрЃў:  
   Рђб РюЁ рЃарЃћрЃњрЃўрЃАрЃбрЃарЃљрЃфрЃўрЃўрЃА рЃцрЃЮрЃарЃЏрЃљ (username, email, password)  
   Рђб РюЁ рЃљрЃЋрЃбрЃЮрЃарЃўрЃќрЃљрЃфрЃўрЃўрЃА рЃцрЃЮрЃарЃЏрЃљ (email, password)  
   Рђб РюЁ рЃарЃЮрЃџрЃћрЃЉрЃўрЃА рЃЊрЃљрЃцрЃБрЃњрЃюрЃћрЃЉрЃќрЃћ рЃгрЃЋрЃЊрЃЮрЃЏрЃўрЃА рЃЎрЃЮрЃюрЃбрЃарЃЮрЃџрЃў (Admin рЃЊрЃљ User рЃарЃЮрЃџрЃћрЃЉрЃў)  
   Рђб РюЁ рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃЌрЃљрЃюрЃЌрЃљ рЃАрЃўрЃљ РђЊ рЃАрЃўрЃљрЃА рЃ«рЃћрЃЊрЃљрЃЋрЃА рЃарЃЮрЃњрЃЮрЃарЃф рЃљрЃЊрЃЏрЃўрЃюрЃўрЃАрЃбрЃарЃљрЃбрЃЮрЃарЃў, рЃљрЃАрЃћрЃЋрЃћ рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃћрЃџрЃў.  
   Рђб РюЁ рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃћрЃџрЃЌрЃљ рЃАрЃўрЃљ рЃЏрЃЮрЃЦрЃЏрЃћрЃЊрЃћрЃЉрЃћрЃЉрЃўрЃА рЃдрЃўрЃџрЃљрЃЎрЃћрЃЉрЃўрЃЌ - рЃЊрЃћрЃбрЃљрЃџрЃћрЃЉрЃўрЃА рЃюрЃљрЃ«рЃЋрЃљ, рЃарЃћрЃЊрЃљрЃЦрЃбрЃўрЃарЃћрЃЉрЃљ(рЃљрЃЊрЃЏрЃўрЃюрЃўрЃАрЃбрЃарЃљрЃбрЃЮрЃарЃў), рЃгрЃљрЃерЃџрЃљ(рЃљрЃЊрЃЏрЃўрЃюрЃўрЃАрЃбрЃарЃљрЃбрЃЮрЃарЃў).  
   Рђб РюЁ рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃўрЃА рЃЊрЃћрЃбрЃљрЃџрЃћрЃЉрЃў РђЊ рЃЊрЃћрЃбрЃљрЃџрЃћрЃЉрЃўрЃА рЃюрЃљрЃ«рЃЋрЃљ рЃАрЃ«рЃЋрЃљ рЃарЃљрЃБрЃбрЃќрЃћ рЃњрЃљрЃЊрЃљрЃАрЃЋрЃџрЃўрЃЌ.  
   Рђб РюЁрЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃўрЃА рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃљ - рЃљрЃЊрЃЏрЃўрЃюрЃўрЃАрЃбрЃарЃљрЃбрЃЮрЃарЃА рЃерЃћрЃБрЃФрЃџрЃўрЃљ рЃЊрЃљрЃљрЃЏрЃљрЃбрЃЮрЃА рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃћрЃџрЃў рЃЌрЃљрЃЋрЃўрЃАрЃў рЃљрЃюрЃњрЃљрЃарЃўрЃерЃўрЃЊрЃљрЃю.  
   Рђб РюЁ рЃАрЃљрЃЎрЃБрЃЌрЃљрЃарЃў рЃърЃљрЃарЃЮрЃџрЃўрЃА рЃерЃћрЃфрЃЋрЃџрЃљ (рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃћрЃџрЃў, рЃљрЃЊрЃЏрЃўрЃюрЃўрЃАрЃбрЃарЃљрЃбрЃЮрЃарЃў),  
   Рђб РюЁ рЃАрЃ«рЃЋрЃўрЃАрЃў рЃърЃљрЃарЃЮрЃџрЃўрЃА рЃерЃћрЃфрЃЋрЃџрЃљ (рЃљрЃЊрЃЏрЃўрЃюрЃўрЃАрЃбрЃарЃљрЃбрЃЮрЃарЃў)

2. Route Guards рЃЊрЃљ рЃгрЃЋрЃЊрЃЮрЃЏрЃўрЃА рЃЎрЃЮрЃюрЃбрЃарЃЮрЃџрЃў  
   Рђб РюЁ Route Guard-рЃћрЃЉрЃў рЃЊрЃљ UI-рЃќрЃћ рЃгрЃЋрЃЊрЃЮрЃЏрЃўрЃА рЃерЃћрЃќрЃдрЃБрЃЊрЃЋрЃљ. рЃљрЃЋрЃбрЃЮрЃарЃўрЃќрЃљрЃфрЃўрЃўрЃА рЃњрЃљрЃарЃћрЃерЃћ рЃерЃћрЃАрЃЋрЃџрЃљ рЃЋрЃћрЃа рЃЏрЃЮрЃ«рЃћрЃарЃ«рЃЊрЃћрЃЉрЃљ рЃЋрЃћрЃарЃфрЃћрЃарЃЌ рЃарЃЮрЃБрЃбрЃќрЃћ.

3. Angular & TypeScript  
   Рђб РюЁ рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃБрЃџрЃўрЃљ рЃБрЃљрЃ«рЃџрЃћрЃАрЃў Angular 19 рЃЊрЃљ TypeScript.  
   Рђб РюЁ рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃљ рЃљрЃгрЃДрЃЮрЃЉрЃўрЃџрЃўрЃљ рЃЎрЃЮрЃЏрЃърЃЮрЃюрЃћрЃюрЃбрЃћрЃЉрЃАрЃљ рЃЊрЃљ рЃАрЃћрЃарЃЋрЃўрЃАрЃћрЃЉрЃќрЃћ рЃЊрЃљрЃДрЃарЃЊрЃюрЃЮрЃЉрЃўрЃЌ.

4. рЃцрЃЮрЃарЃЏрЃћрЃЉрЃў & рЃюрЃљрЃЋрЃўрЃњрЃљрЃфрЃўрЃљ  
   Рђб РюЁ рЃарЃћрЃњрЃўрЃАрЃбрЃарЃљрЃфрЃўрЃўрЃАрЃљ рЃЊрЃљ рЃљрЃЋрЃбрЃЮрЃарЃўрЃќрЃљрЃфрЃўрЃўрЃА рЃцрЃЮрЃарЃЏрЃћрЃЉрЃў рЃљрЃњрЃћрЃЉрЃБрЃџрЃўрЃљ Reactive Forms-рЃўрЃЌ.  
   Рђб РюЁ рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃБрЃџрЃўрЃљ Angular Router рЃњрЃЋрЃћрЃарЃЊрЃћрЃЉрЃА рЃерЃЮрЃарЃўрЃА рЃњрЃљрЃЊрЃљрЃАрЃљрЃАрЃЋрЃџрЃћрЃџрЃљрЃЊ + lazy loading.  
   Рђб РюЁ рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃБрЃџрЃўрЃљ shared рЃЎрЃЮрЃЏрЃърЃЮрЃюрЃћрЃюрЃбрЃћрЃЉрЃў рЃарЃћрЃњрЃўрЃАрЃбрЃарЃљрЃфрЃўрЃљ-рЃљрЃЋрЃбрЃЮрЃарЃўрЃќрЃљрЃфрЃўрЃљрЃќрЃћ & рЃАрЃљрЃЎрЃБрЃЌрЃљрЃарЃў рЃЊрЃљ рЃАрЃ«рЃЋрЃўрЃАрЃў рЃърЃљрЃарЃЮрЃџрЃўрЃА рЃфрЃЋрЃџрЃўрЃџрЃћрЃЉрЃљрЃќрЃћ.

5. HTTP рЃЎрЃЮрЃЏрЃБрЃюрЃўрЃЎрЃљрЃфрЃўрЃљ рЃЊрЃљ Backend  
   Рђб РюЁ рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃБрЃџрЃўрЃљ HttpClient рЃАрЃћрЃарЃЋрЃћрЃарЃЌрЃљрЃю рЃЊрЃљрЃАрЃљрЃЎрЃљрЃЋрЃерЃўрЃарЃћрЃЉрЃџрЃљрЃЊ  
   Рђб РюЁ back-end рЃџрЃЮрЃњрЃўрЃЎрЃљ рЃЏрЃЮрЃЌрЃљрЃЋрЃАрЃћрЃЉрЃБрЃџрЃўрЃљ mock-server рЃАрЃљрЃЦрЃљрЃдрЃљрЃџрЃЊрЃћрЃерЃў. рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃБрЃџрЃўрЃљ json-server рЃЊрЃљ Custom Express Middleware (server.js):  
    /login, /register, /changePassword, /admin/changePassword, /promote  
   Рђб РюЁ рЃърЃљрЃарЃЮрЃџрЃўрЃА рЃфрЃЋрЃџрЃўрЃџрЃћрЃЉрЃўрЃА рЃЊрЃљ рЃарЃЮрЃџрЃўрЃА рЃЏрЃљрЃарЃЌрЃЋрЃўрЃА рЃџрЃЮрЃњрЃўрЃЎрЃљ back-end-рЃќрЃћ.

6. рЃерЃћрЃфрЃЊрЃЮрЃЏрЃћрЃЉрЃўрЃА рЃЊрЃљрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃљ & UX  
   Рђб РюЁ рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃБрЃџрЃўрЃљ рЃљрЃюрЃњрЃБрЃџрЃљрЃа рЃЏрЃљрЃбрЃћрЃарЃўрЃљрЃџрЃўрЃА SnackBar-рЃћрЃЉрЃў рЃгрЃљрЃарЃЏрЃљрЃбрЃћрЃЉрЃўрЃА/рЃерЃћрЃфрЃЊрЃЮрЃЏрЃўрЃА рЃерЃћрЃбрЃДрЃЮрЃЉрЃўрЃюрЃћрЃЉрЃћрЃЉрЃўрЃА рЃњрЃљрЃЏрЃЮрЃАрЃљрЃбрЃљрЃюрЃљрЃЊ.  
   Рђб РюЁ Tooltip-рЃћрЃЉрЃў рЃерЃћрЃќрЃдрЃБрЃЊрЃБрЃџрЃў рЃгрЃЋрЃЊрЃЮрЃЏрЃўрЃА рЃдрЃўрЃџрЃљрЃЎрЃћрЃЉрЃќрЃћ.

7. Signals & Change Detection  
   Рђб РюЁ рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃБрЃџрЃўрЃљ Angular Signals рЃАрЃЌрЃћрЃўрЃЌрЃўрЃА рЃАрЃљрЃЏрЃљрЃарЃЌрЃљрЃЋрЃљрЃЊ.  
   Рђб РюЁ рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃБрЃџрЃўрЃљ OnPush ChangeDetection рЃърЃћрЃарЃцрЃЮрЃарЃЏрЃљрЃюрЃАрЃўрЃАрЃЌрЃЋрЃўрЃА.

8. рЃАрЃбрЃўрЃџрЃў рЃЊрЃљ рЃЌрЃћрЃЏрЃљ  
   Рђб РюЁ рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃБрЃџрЃўрЃљ TailwindCSS UI-рЃўрЃАрЃЌрЃЋрЃўрЃА.  
   Рђб РюЁ рЃўрЃюрЃбрЃћрЃарЃцрЃћрЃўрЃАрЃў рЃљрЃгрЃДрЃЮрЃЉрЃўрЃџрЃўрЃљ Angular Material-рЃўрЃЌ.

9. Responsiveness & UX

10. рЃАрЃљрЃБрЃЎрЃћрЃЌрЃћрЃАрЃЮ рЃърЃарЃљрЃЦрЃбрЃўрЃЎрЃћрЃЉрЃў

    Рђб РюЁ рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃБрЃџрЃўрЃљ shared & re-usable рЃЎрЃЮрЃЏрЃърЃЮрЃюрЃћрЃюрЃбрЃћрЃЉрЃў
